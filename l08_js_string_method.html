자바스크립트 문자열 String의 메소드

문자열은 유저에게 전달하는 데이터로 웹상에서 가장 많이 사용된다. 때문에 문자열을 다루는 함수가 많이 존재한다.

method 설명 예시
toUpperCase(),toLowerCase() 문자열의 대문자 or 소문자로 반환 "AbCdEFgh".toUpperCase():
"AbCdEFgh".toLowerCase():
trim() 문자열의 앞뒤 공백을 제거한다. " 이 창영 ":.trim().length:
charAt(),charCodeAt() charAt()은 문자열에서 특정 인덱스에 위치하는 유니코드 단일문를 반환
charCodeAt() 주어진 인덱스에 대한 UIF-16코드를 반환		 "ABCDE".charCodeAt(0):
"ABCDE".charAt(0):
"ABCDE"[0]:
slice(start,end) 일부 문자를 추출할 때 사용한다. "스파게티 피자 된장찌개 샌드위치".slice(8,12) :
"스파게티 피자 된장찌개 샌드위치".slice(-9,-5) :
"스파게티 피자 된장찌개 샌드위치".slice(5) :
"스파게티 피자 된장찌개 샌드위치".slice(-5) :
foodList.slice(foodList.length-4,foodList.length) :
substring(start,end) slice와 동일하지만 양수 인덱스만 사용가능 "스파게티 피자 된장찌개 샌드위치".substring(5,7) :
substr(start, length) 일부 문자를 추출한다. 이 때 시작점부터 길이까지 추출한다. "스파게티 피자 된장찌개 샌드위치".substr(8,4) :
"스파게티 피자 된장찌개 샌드위치".substr(-9,4) :
"스파게티 피자 된장찌개 샌드위치".substr(-9,length-4) :
startsWith(string, position) 특정 문자열로 시작하는지 여부(boolean) *굳이 position 넣어야하나? foodList.startsWith("스파") :
foodList.startsWith("된장",8) :
endswith(String,position=String.length) 끝에서부터 특정 문자열로 시작하는지 여부(boolean) foodList.endsWith("위치") :
foodList.endsWith("샌드") :
foodList.endsWith("된장") :
includes(String,start position=0) 특정 위치부터 특정 문자열이 있는지 여부 검색 (boolean) foodList.includes("찌개") :
foodList.includes("찌개",11) :
indexOf(String,start position=0) 시작 또는 특정 위치 부터 특정 문자열이 있는지 검색 후 해당 문자열의 시작지점을 반환, 특정 문자열이 없을 때 -1을 반환 foodList.indexOf("피자") :
foodList.indexOf("피자",6) :
lastIndexOf(String,start position=0) 끝 또는 특정 위치부터 거꾸로 특정 문자열이 있는지 검색 후 해당 문자열의 시작지점을 반환, 특정 문자열이 없을 때 -1을 반환 "Brave new world. Brave!".lastIndexOf("r") :
"Brave new world. Brave!".indexOf("r") :
"Brave new world. Brave!".lastIndexOf("Brave") :
"Brave new world. Brave!".indexOf("Brave") :
"Brave new world. Brave!".lastIndexOf("new",5) :
"Brave new world. Brave!".indexOf("new",5) :
split(String, limit length) 특정 문자열을 기준으로 나누어 배열로 만든다.(길이를 정할 수 있다.) foodList.split(" ",3) :
foodList.split(",") :
replace() 특정 문자열이나 정규식 패턴으로 검색 후 다른 문자열로 변환한 값을 반환 (x=x.replacE() x, x != x.replace()) // 실제 값을 변경시키는게 아님 foodList.replace("된장찌개","스테이크") :
foodList.replace(" ",",") :
replaceAll() 특정 문자열이나 정규식 패턴으로 검색 후 검색한 모든 문자열을 다른 문자열로 변환한 값을 반환 foodList.replaceAll(" ",",") :
foodList.replaceAll(" ",",") :
