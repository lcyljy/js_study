문자들의 집합 문자열 String

String 객체는 문자들의 집합인 문자열을 조작하는 함수를 제공한다.

문자열 생성 방법

  • let x="abc";(기본데이터 타입이다)
  • let x='abc';(자바스크립트는 char 캐릭터 타입이 없다.)
  • let x='abc';(리터럴 템플릿 적용 문자)

문자(char) 접근하기

  • "string"[2] :
    * 배열처럼 보이게 할 수는 있지만 배열처럼 바꾸는 것은 불가능 ex. "string"[2] ="a"; (x)
  • "string".charAt(2) :

기본 데이터 타입인 문자열과 인스턴트 객체인 문자열

  • let stringVal="안녕하세요"; (기본 데이터 타입인 문자열)
  • let objectVal=new String("안녕하세요"); (new 연산자로 생성한 인스턴트 객체)
  • typeof stringVal :
  • typeof objectVal :
  • stringVal==objectVal: (value는 같다)
  • stringVal===objectVal: (type은 다르다)
  • 기본데이터 타입인 문자열에는 함수가 존재하지 않는다. 하지만 사용할 수 있는 이유는 사용할 때 인스턴트 객체 생성과 비슷한 일이 발생하기 때문이다.

이스케이프 시퀸스

특수문자나 백슬러시, 줄 바꿈, 따옴표 같은 것을 출력할 수 있다.

  • \"
  • \'
  • \\
  • \n
  • \t
  • \b

템플릿 리터럴 ${표현식}

문자열을 ``백틱으로 선언하면 템플릿 리터럴 ${}표현식을 사용한다는 선언이다. 이 때 자바스크립트 문법을 표현식에 사용할 수 있다.
