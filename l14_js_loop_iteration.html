자바 스크립트의 루프와 반복

for,forin,forof,while,dowhile 반복문(iteration statements)이 있다.

똑같은 명령을 일정 횟수만큼 반복하기 위해 작성되는 문법, 대체로 검사식에 많이 이용.

for 반복문

for([초기문]; [조건문]; [증감문];{반복실행} : for는 조건이 true일 때 반복실행한다

  • [11,2,20,3,13,1,0,-20]의 max :
  • [11,2,20,3,13,1,0,100,-20] 높은 순서대로 정렬 :
  • [11,2,-20,3,13,1,0,100,20] 낮은 순서대로 버블정렬(밀어내기) :
  • 높은 순서대로 선택정리하세요

forin 반복문

for(key in iterationObject){} : key를 가진 객체 key를 반환할 수 없을 때까지 반복을 실행

JSON{key:value,key:value} => key를 반환, Array[a,b,c] => index를 반환

  • ["a","b","c"] :
  • {name:"이창영",age=28,email="lodin_love@naver.com"}
  • "abcdefg" :

forof 반복문

for (object of iterationObject){}

반복 가능한 데이터 타입의 객체에서 더이상 내용물이 나오지 않을 때까지 반복실행

(iterationObject).next(): next()는 포인트가 객체 밖에 있다가 내용물이 있으면 true를 반환하고 내용물로 포인트를 옮긴다. 그리고 내용물도 반환한다. 만약 값이 더이상 나오지 않으면 false를 반환한다.

배열, Map(json과 비슷), Set(배열인데 복수의 값을 제거), 문자열

key값을 알아내기 힘들다

  • ["a","b","c"] :
  • {name: "이창영", age: 28, email: "lodin_love@naver.com"} : json은 next()가 지원하지 않는다.
  • "abcdefg" :
  • {black:["black","white"],default:["white","black"],darkgray:["darkgray","bisque"],pink["lavender","indigo"]} :
