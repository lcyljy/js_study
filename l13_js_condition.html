자바스크립트의 조건문

블럭 안의 실행을 조건에 따라 동적으로 선택되게 하는 조건문 if, switch, 삼항연산자

if 조건문

if(condition){ 블럭 } : 조건이 참이면(true) 블럭을 실행한다.

if(){}else{} : if의 조건이 맞지 않으면 else의 블럭을 실행한다.

if(condition1){}else if(condition2{}else{} : 1번 조건이 만족하지 않으면 2번 조건(else if) 실행, 1번과 2번 모두 만족하지 않으면 else 실행)

  • if(value < 50){} 값 구하기
  • 50보다 작고 20보다 큰 수 값 구하기
  • 50보다 크거나 20보다 작은 수 입력
  • 50보다 크거나 20보다 작은 수 입력

Switch 조건문

        switch(표현식or변수){
          case a:
            실행
          break;
          case b:
            실행
          break;
          case c:
            실행
          break;
          default:
            실행
        }

스위치 케이스는 엄격한 비교(===)를 사용하고 위에서부터 일치하는 항목을 찾으면 switch는 종료된다. (엄격한 비교 외의 조건을 사용할 수 없다.)

if에서 else if 문과 동일하게 작동하지만 조건문의 수행 속도가 굉장히 빠르다

조건을 파악하기 굉장히 좋다.

삼항연산자

작성법1: (condition)? true : false == if(condition){true}else{false}

작성법2: let variable = (condition) ? trueObject : falseObjcet;

작성법3:
          (condition) ? true
        : (condition2) ? true
        : (condition3) ? true
        : false

if(condition){}else if(condition2){}else if(condition3)?{}else{}

조건 파악이 편하고 코드가 간결해 프로그래밍 속도를 증가시킨다.
