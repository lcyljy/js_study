자바스크립트의 배열

여러 변수를 index로 참조해 저장하는 데이터 타입(Iterator 상속하고 있다.)

작성법 : const array =[item1, item2, ....] (기본데이터타입 처럼 사용가능)

작성법2 : const array=new Array(item1, imes2, .....) (권장하진 않음)

호출법: array[index]

대입 : array[index]=value

["a","b","c","d"].toString(): (객체를 바로 호출하면 toString()이 생략된다.)

대입: array[index]=value 예) a[1]="B" :

typeof []: (배열의 타입을 typeof로 검사할 수 없다.)

Array.isArray([]): (Array 타입이면 true)

Array[maxIndex+1]=value: (자바스크립트는 동적으로 배열의 길이를 조절한다)
