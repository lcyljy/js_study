자바스크립트 배열의 유용한 메소드

const a=["b","c","a","z","f","e","h"];

  • a.toString(): 쉼표로 구분된 배열의 값을 문자열로 출력한다.
  • a.join() : 배열의 값을 join()으로 결합한 문자열을 출력한다
  • a.pop() : 배열의 마지막 값을 제거하면서 반환한다.
  • a.push() : 배열의 마지막에 새 값을 추가한다.
  • a[a.length]="k" : 배열의 마지막에 새 값을 추가한다.
    *(a[maxIndex+1==a.length])
  • a.shift() : 배열의 첫번째 값을 제거하고 반환한다.(모든 인덱스를 한칸 당긴다)
  • a.unshift() : 배열의 첫번째에 새 값을 추가한다. (모든 인덱스가 한칸 밀린다.)
  • a.concat(9,0,[1,3,2],[4]) : 배열의 끝에 '새로운 배열을 추가한' 배열을 반환한다.(배열을 바꾸지 않고 다른 배열을 반환한다.) -기존배열은 그대로
  • a.slice(start_index, end_index) : 특정 위치(start~end)의 배열을 새로운 배열로 반환한다. - 기존배열은 그대로
  • a.splice(start, delete_length, insert_item...) : 특정 위치부터 원하는 길이까지 삭제하고 추가할 수 있다.



  • a.sort() : 배열의 요소를 순서대로 정렬한다.
  • a.reverse() : 배열의 요소를 역순으로 정렬한다.
