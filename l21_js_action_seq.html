  • ul>li 노드에 mouseenter를 재정의한다.

HTML 문서에 선언된 스크립트의 동작 순서

  1. 브라우저는 html 문서를 css와 병합해 DOMTree구조를 생성한다.
  2. 브라우저가 병합된구조를 보고 화면에 그래픽으로 순서대로 개체를 표현한다.(순차적언어)
  3. 개체를 출력하던 중 자바스크립트 구문을 발견하면 실행한다
  4. 이 때 자바스크립트가 노드 개체를 검색하던 중이면 브라우저가 해당 노드 개체를 출력해야 찾을 수 있다.

순차적 언어인 html과 자바스크립트가 문제를 일으키지 않게 하는 방법

  • 자바스크립트를 모든 노드가 생성된 시점이 body 아래에 작성한다.(링크를 한다.)
  • onload : 해당하는 노드가 생성되었는지 확인 / window.onload=function(){스크립트 작성}
  • window 객체가 완전히 생성된 시점 즉 노드가 모두 로드된 시점에
