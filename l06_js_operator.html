할당 연산자와 증가 및 감소 연산자

operator(연산) 이름 설명 예시
a+=b 추가할당 a=a+b : 자기 자신 a에 b를 더한다 x+=10 (x=x+10)
a-=b 빼기할당 a=a-b : 자기 자신 a에 b를 뺀다 x-=10 (x= x-10)
a*=b 곱셈할당 a=a*b : 자기 자신 a에 b를 곱한다 x*=3 (x=x*3)
a/=b 분할할당 a=a/b : 자기 자신 a에 b를 나눈다 x/=7 (x=x/7)
++a; or --a; 증가 및 감소 연산자 a=a+1; or a=a-1; 숫자 앞에 증가 및 감소연산자가 오면 연산을 먼저하고 다른 실행을 옮긴다 a(++i); : i를 1증가시키고 a()를 실행한다
a++; or a--; 증가 및 감소 연산자 a=a+1; or a=a-1; 숫자 뒤에 증가 및 감소연산자가 오면 실행을 먼저하고 연산을 해라 a(i++); : a()를 실행시키고 i를 1증가시킨다

비교연산자

operator(연산) 이름 설명 예시
== 등호
!= 비 평등비교 a와 b의 값이 같지 않은지 비교 10! = (6+4) :
=== 엄격한 평등 a와 b가 (타입까지) 완전히 동일한지 바교 10===new Number(10) :

10=== "10"
!== 엄격한 비동등 a와 b가 (타입까지) 완전히 동일하지 않은지 바교 10!==new Number(10) :

10!== "10"
a<b 미만(작다) 왼쪽 값 a가 오른쪽 값 b보다 작은지 확인 5<3 : false
a>b 초과(크다) 왼쪽 값 a가 오른쪽 값 b보다 큰지 확인 5>3 : true
a<=b 이하 왼쪽 값 a가 오른쪽 값 b보다 같거나 작은지 확인 5<=3 : false
a>=b 이상 왼쪽 값 a가 오른쪽 값 b보다 같거나 큰지 확인 5>=3 : true
